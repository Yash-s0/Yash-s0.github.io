URL Shortener (API + Frontend)

Fully featured URL shortener with JWT authentication, SQLAlchemy storage, and a complete custom frontend UI.

Skills Used
Python Flask SQLAlchemy JWT JavaScript HTML CSS Git passlib
Backend Code Frontend Code

Features

  • User registration & login (JWT)
  • Stores all URLs + user search history
  • FastAPI backend + SQLAlchemy ORM storage
  • Full custom frontend UI built from scratch

How it works

FastAPI provides secure endpoints for registering users, logging in, and shortening URLs. SQLAlchemy stores the mapping of long → short URLs and user history. The frontend communicates with the backend via JWT-authenticated API calls.
