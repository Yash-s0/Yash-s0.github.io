Contacts API (CRUD)

Lightweight FastAPI contacts manager — create, update, delete, search and list contacts.

Skills used
Python FastAPI JSON Git
Source Code

Features

  • Create or update contacts
  • Delete stored contacts
  • Search contacts
  • List all contacts

How it works

FastAPI exposes REST endpoints. For demo simplicity, persistence is a JSON file. Endpoints validate input and persist to disk asynchronously.
