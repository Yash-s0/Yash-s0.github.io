Etch-A-Sketch Canvas App

A browser-based sketching app built with the HTML Canvas API. Allows drawing, erasing, clearing, and customizing brush size & color.

Skills used
JavaScript HTML CSS Canvas API Git
Play Online Source Code

Features

  • Live drawing using Canvas 2D context
  • Brush size & color selection
  • Eraser tool
  • Clear canvas feature
  • Responsive layout across devices

How it works

Mouse (or touch) events are captured on the canvas. A 2D context draws lines by tracking pointer movement. Tools like eraser and brush resize dynamically modify the stroke settings. Reset clears the rendering context fully.
