Binance 24h Crypto Tracker

Real-time cryptocurrency tracker using Binance API. Displays price, 24h high/low, and updates live through a lightweight Flask backend.

Skills used
Python Flask Binance API JavaScript HTML CSS Git
Source Code

Features

  • Live real-time updates using Binance API
  • Displays 24h high, low and volume
  • Lightweight polling using Flask backend
  • Clean and responsive UI

How it works

The Flask backend polls Binance public endpoints every few seconds (rate-limited). It stores minimal snapshot values and serves lightweight JSON for fast client rendering. The frontend updates prices using fetch/WebSocket without reload.
