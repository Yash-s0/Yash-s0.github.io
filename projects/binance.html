Binance 24h Crypto Tracker

Real-time cryptocurrency tracker using Binance API. Displays price, 24h high/low, and updates live through a lightweight Flask backend.

Python Flask Binance API JavaScript HTML CSS Git
Source Code

Problem

Needed a lightweight way to monitor 24h crypto movement without a heavy dashboard.

Constraints

Keep it fast, simple, and low-cost with minimal backend state.

Architecture / Decisions

Flask backend polls Binance public endpoints with rate limits in mind, serves minimal JSON snapshots, and the frontend updates prices without page reloads for a fast UX.

My Contribution

  • Built the Flask API and polling flow.
  • Implemented the real-time UI updates.
  • Designed a clean, responsive layout.

Outcome / Impact

Impact not formally measured yet. The project provides a low-latency market snapshot and a clean demo of API + UI flow.
