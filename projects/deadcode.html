PyDeadCode — Dead Code Detector

Open-source Python 3.8+ static analyzer that finds unused imports, functions, classes, variables, and unreachable code, and ships a polished, self-contained HTML report for quick refactors. (Dec 2025)

Python Static Analysis AST Jinja2 HTML CSS CLI Open Source
Source Code

Problem

Large Python codebases accumulate dead code and unused imports, making refactors risky and review cycles slow.

Constraints

Needed a single-file report that works locally without a server and avoids false positives across modules.

Architecture / Decisions

Walks the project tree, parses each .py file into an AST, and builds a call graph to map definitions to references. Unreachable blocks (post-return/raise) are classified separately and rendered into a self-contained HTML report via Jinja2.

My Contribution

  • Designed AST parsing flow and project-wide symbol tracking.
  • Implemented call graph linking to reduce cross-module false positives.
  • Shipped the HTML report layout and search/collapse UX.

Outcome / Impact

Impact not formally measured yet. The tool provides a repeatable audit workflow for cleanup and refactors.
