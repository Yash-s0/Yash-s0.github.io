PyDeadCode — Dead Code Detector

Open-source Python 3.8+ static analyzer that finds unused imports, functions, classes, variables, and unreachable code, and ships a polished, self-contained HTML report for quick refactors. (Dec 2025)

Skills Used
Python Static Analysis AST Jinja2 HTML CSS CLI Open Source
Source Code

Features

  • Detects unused imports, functions/classes, variables, and unreachable code across a Python project.
  • Builds a project-wide call graph to avoid false positives on cross-module usage.
  • Computes a single "health score" summarizing dead-code density.
  • Outputs a responsive, keyboard-friendly HTML report with search and collapsible sections.

How it works

PyDeadCode recursively walks a target directory, parses each .py file into an AST, and tracks definitions, imports, and references. It then builds a call graph to understand which symbols are truly unused, classifies unreachable segments (e.g., after return or raise), and renders the aggregated findings into a single-page HTML report using a bundled Jinja2 template.
