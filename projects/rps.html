Rock Paper Scissors Game

A browser-based Rock Paper Scissors game with animations, scoring, and modern UI — fully implemented in JavaScript.

Skills used
JavaScript HTML CSS Git
Play Online Source Code

Features

  • Interactive, fast gameplay
  • Smooth animations for win/lose/draw
  • Dynamic scoring system
  • Frontend-only (no backend required)
  • Hosted on GitHub Pages

How it works

The game runs entirely in the browser using JavaScript event listeners. User actions (Rock / Paper / Scissors) are compared against a randomly generated computer choice. DOM updates then animate score changes and round results.
