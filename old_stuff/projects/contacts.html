Contacts API (CRUD)

Lightweight FastAPI contacts manager — create, update, delete, search and list contacts.

Release: 2024
Type: API
Language: Python

Source Code

Problem

Needed a simple, testable CRUD API to manage contact data.

Challenges

Keep the storage minimal and the API easy to reason about.

Architecture Overview

FastAPI exposes REST endpoints with input validation. For simplicity, persistence uses a JSON file, keeping the data layer transparent and easy to inspect.
